# Η αποστολή του Learn LaTeX

Αυτός ο ιστότοπος έχει ως στόχο να σας βοηθήσει να κάνετε τα πρώτα βήματά σας με το LaTeX, ένα σύστημα προετοιμασίας εγγράφων σχεδιασμένο για να παράγει υψηλής ποιότητας στοιχειοθετημένα αποτελέσματα.
Το LaTeX μπορεί να τρομάζει τους νέους χρήστες καθώς δεν είναι ένας επεξεργαστής κειμένου, και γιατί δεν είναι ένα ενιαίο πρόγραμμα. 
Στόχος μας είναι να σας βοηθήσουμε να ξεκινήσετε χωρίς να προσπαθούμε να σας δείξουμε τα πάντα μονομιάς.

Για να το πετύχουμε, πήραμε περίπου 16 από τα πιο σημαντικά στοιχεία που θα χρειαστεί να γνωρίζετε, και φτιάξαμε με αυτά σύντομα «μαθήματα». Κάθε μάθημα είναι πολύ εστιασμένο, επομένως δεν θα χρειαστείτε πολύ χρόνο για να τα καλύψετε. Σε κάθε μάθημα δίνουμε παραδείγματα για το τι θα γράφατε, και μπορείτε να τα δοκιμάσετε άμεσα μέσα στον ιστότοπο. Μάθετε περισσότερα σχετικά με το [πώς δουλεύουν τα παραδείγματα](./help#examples) στην ενότητα της βοήθειας.
